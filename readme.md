## Dependencies

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install:

```bash
python3 -m pip install numpy
python3 -m pip install matplotlib
python3 -m pip install tsplib95
```

## Usage

```python3 run.py [input_data_name.tsp]```
